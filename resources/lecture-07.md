**Error Types Reference:**

- **Error**: Generic errors and custom messages
- **TypeError**: Type-related errors (wrong data types)
- **ReferenceError**: Undefined variables or functions
- **RangeError**: Values outside expected ranges
- **SyntaxError**: Code syntax problems (less common in runtime)
